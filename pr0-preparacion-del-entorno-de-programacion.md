# PR0 - Preparación del entorno de programación

## ¿Con qué entorno debo desarrollar?

El editor que debes emplear para desarrollar las prácticas es el CodeLite.

Aunque puedes instalar el editor CodeLite en tu máquina física, a fin de evitar problemas con la instalación y configuración de CodeLite, y los ocasionados por las diferencias entre compiladores C según el sistema operativo, te proporcionamos una máquina virtual con el entorno instalado. Accede a la máquina virtual desde el aula: _Apartado Recursos -&gt; Materiales -&gt; Máquina virtual -&gt; Software en linea -&gt; descargar FP20172.ova_ y sigue las siguientes instrucciones para instalarla en tu ordenador y empezar a utilizar el CodeLite: [https://xwiki.recursos.uoc.edu/wiki/mat05555cat/view/FP/Introducci%C3%B3 a l%27entorn de programaci%C3%B3 Codelite/](https://xwiki.recursos.uoc.edu/wiki/mat05555cat/view/FP/Introducci%C3%B3%20a%20l%27entorn%20de%20programaci%C3%B3%20Codelite/)

## Cuando trato de hacer el Build y ejecutar el código de la PR0, aparecen errores

Te recomiendo que purgues el proyecto con la opción Clean antes de hacer el Build.

## Cuando trato de ejecutar la PR0, tanto en modo Release como en Debug, tienen el mismo comportamiento

Cuando quieras cambiar la ejecución entre diferentes configuraciones del proyecto \(en la PR0 dispones de Release y de Debug\), recuerda siempre volver a hacer el Build de la configuración en cuestión.
