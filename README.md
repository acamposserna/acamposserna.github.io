# Introducción

Esta página es un recopilatorio de las preguntas planteadas en el foro de la asignatura de **Laboratorio de Prácticas de Programación** durante el presente semestre.

Las preguntas están agrupadas por Práctica. El documento se irá actualizando regularmente con las nuevas preguntas planteadas en el foro.
